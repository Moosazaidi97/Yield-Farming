# 🌾 Yield Farming (Advanced DeFi)

Multi-pool yield farming smart contract.

## ✨ Features
- Multiple pools
- Reward per block
- Deposit / Withdraw
- Pending rewards

## 🛠 Tech
- Solidity
- Ethers.js

## ▶️ Usage
1. Deploy reward token
2. Deploy farm contract
3. Add pools
4. Users deposit tokens

## ⚠️ Important
- Contract must hold reward tokens
- Allocation points control rewards
